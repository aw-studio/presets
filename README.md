# AW-Studio Presets

## Installation

```bash
composer require aw-studio/presets
```

## Presets

### Vue.js
```shell
php artisan install:vue
```
### Typescript
```shell
php artisan install:ts
```
### Inertia
```shell
php artisan install:inertia
```

## Bundles

### Admin
The admin bundle includes Typescript, Inertia, macrame
```shell
php artisan install:admin
```
